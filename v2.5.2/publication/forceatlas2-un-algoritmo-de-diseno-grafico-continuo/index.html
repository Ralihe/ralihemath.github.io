ForceAtlas2, un algoritmo de diseño gráfico continúo: Aplicación para el estudio de las élites

González-Bustamante, B., Cisternas, C.
November 2018
DOI

Abstract

This working paper tests an alternative algorithm for the legislative periods analysed by González-Bustamante and Cisternas (2016). Specifically, ForceAtlas2 is used, which is a continuous graph layout algorithm developed by Jacomy, Venturini, Heymann, and Bastian (2013) based on a force-directed design.
